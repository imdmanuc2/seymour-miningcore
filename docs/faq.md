# Frequently Asked Questions

## Why is Expected Solo Time so high?

Solo mining is probabilistic. Expected time is a long-term statistical
average.

## Why does hashrate fluctuate?

MiningCore estimates hashrate from submitted shares, so values naturally
move.

## Why don't I see my worker?

Verify your wallet address, worker name, and MiningCore API.

## Does this support multiple coins?

The dashboard is designed for multi-coin monitoring.

## Does this work on Raspberry Pi?

Yes. ARM64 is a primary deployment target.
