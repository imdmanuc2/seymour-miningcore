# Repository Map

## Main Repo

This repository is the official Seymour MiningCore product.

It contains:

- MiningCore engine
- Installer
- Scripts
- Config templates
- Local Console UI
- API contract
- Nexus integration docs
- Security/license docs

## Folders

docs/       Project documentation
openapi/    API specification
install/    Installer helpers
scripts/    Admin and Nexus scripts
configs/    Example pool configs
systemd/    Linux service files
ui/         Local Console UI
api/        API layer
src/        Source code changes
tests/      Test scripts
releases/   Release notes/manifests
