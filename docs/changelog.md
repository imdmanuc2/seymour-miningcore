# Changelog

All notable changes to Seymour MiningCore will be documented in this
file.

The project follows a simple versioning strategy during early
development:

-   **Alpha** -- Internal development and experimentation.
-   **Beta** -- Public testing and feedback.
-   **Stable** -- Production-ready releases.

------------------------------------------------------------------------

# \[0.9.0-alpha\] - 2026-07-02

## Overview

First documented internal development milestone.

This release establishes the foundation for Seymour MiningCore as a
modern command center for solo cryptocurrency mining.

------------------------------------------------------------------------

## Added

### Dashboard

-   Modern responsive dashboard
-   Dark theme interface
-   Live pool status
-   Pool hashrate cards
-   Active mining operations
-   Latest block information

### Charts

-   Live Pool Hashrate chart
-   Per-Miner Hashrate chart
-   Interactive chart expansion

### Fleet

-   ASIC Fleet overview
-   Fleet Health Score
-   Live worker monitoring
-   Worker ranking

### Mining Intelligence

-   Probability Simulator
-   Multi-Coin Solo Lottery
-   Block Race Meter
-   Latest Block Preview

### User Experience

-   Educational tooltips
-   Achievement system
-   Solo Hall of Fame
-   TV Mode
-   Block Celebration preview

### Infrastructure

-   ARM64 support
-   Docker deployment
-   MiningCore compatibility
-   Documentation framework

------------------------------------------------------------------------

## Improved

-   Dashboard layout
-   Information hierarchy
-   Responsive design
-   Mining education through contextual help
-   Project documentation

------------------------------------------------------------------------

## Fixed

-   MiningCore ARM64 compatibility
-   ZMQ startup issues
-   Docker runtime dependencies
-   BCH synchronization improvements
-   Worker aggregation improvements

------------------------------------------------------------------------

## Known Limitations

-   Bitcoin support is in development.
-   ASIC telemetry (temperature, fan speed, power) is planned.
-   Multi-user authentication is not yet implemented.
-   Remote fleet management is planned.
