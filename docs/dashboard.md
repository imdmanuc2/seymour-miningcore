# Dashboard Guide

## Overview

The Seymour MiningCore Command Center is designed to give miners an
instant understanding of their mining operation. Every widget has a
purpose and updates in real time using the MiningCore API.

------------------------------------------------------------------------

# Dashboard Layout

The dashboard is organized into logical sections:

1.  Mission Control
2.  Pool Status
3.  Live Charts
4.  ASIC Command Center
5.  Mining Statistics
6.  Probability Tools
7.  Historical Information
8.  TV Mode

------------------------------------------------------------------------

# Dashboard Modules

## Active Mining Operations

Displays every active cryptocurrency currently being mined.

Current support:

-   Bitcoin Cash (BCH)
-   Bitcoin (BTC) *(in development)*
-   Dogecoin (DOGE) *(planned)*
-   DigiByte (DGB) *(planned)*

------------------------------------------------------------------------

## Pool Status

Displays:

-   Online / Offline status
-   Connected peers
-   Current block height
-   Network difficulty
-   Network hashrate
-   Pool hashrate

Purpose:

Provides a quick health check of the mining operation.

------------------------------------------------------------------------

## Live Pool Hashrate Chart

Displays total pool hashrate over time.

Features:

-   Auto-refresh
-   Click to enlarge
-   Historical trend
-   Interactive tooltips

------------------------------------------------------------------------

## Per-Miner Hashrate

Displays the live hashrate of every connected ASIC.

Purpose:

Identify underperforming miners quickly.

------------------------------------------------------------------------

## ASIC Command Center

Shows every connected miner.

Each card displays:

-   Worker Name
-   Current Hashrate
-   Shares per Second
-   Status
-   Fleet Ranking

Future versions will include:

-   Temperature
-   Fan Speed
-   Power Consumption
-   Efficiency
-   Miner IP Address
-   Web UI Launch
-   Restart Controls

------------------------------------------------------------------------

## Fleet Health Score

Provides a quick health rating for the mining operation.

A healthy fleet should show:

-   All miners online
-   Stable hashrate
-   No alerts

------------------------------------------------------------------------

## Multi-Coin Solo Lottery

Calculates estimated solo mining odds for every supported coin.

Shows:

-   Expected Solo Time
-   Daily Chance
-   Weekly Chance
-   Monthly Chance

------------------------------------------------------------------------

## Probability Simulator

Allows miners to understand how changing hashrate affects the
probability of finding a block.

------------------------------------------------------------------------

## Latest Block

Displays the latest known block information.

Includes:

-   Height
-   Reward
-   Timestamp

------------------------------------------------------------------------

## Block Race Meter

Shows how close the network is to finding the next block.

This is a visualization intended to help miners understand blockchain
timing.

------------------------------------------------------------------------

## Solo Hall of Fame

Displays historical blocks found by Seymour MiningCore users.

Future versions may include:

-   Wallet
-   Worker
-   Reward
-   Block Height
-   Date Found

------------------------------------------------------------------------

## Achievement System

Awards milestones as miners grow.

Examples:

-   First ASIC
-   First TH/s
-   First Block
-   100 Days Online

------------------------------------------------------------------------

## TV Mode

Optimized for wall-mounted displays.

Features:

-   Large fonts
-   High visibility
-   Automatic updates
-   NOC-style layout

------------------------------------------------------------------------
