# Configuration

## Core Settings

-   Dashboard Port
-   MiningCore API URL
-   Refresh Interval
-   Theme
-   Supported Coins
-   Wallet Address
-   Pool Fee
-   Developer Fee

## Multi-Coin

Supported design:

-   BCH
-   BTC
-   DOGE
-   DGB

Additional coins can be added in future releases.

## Notes

Use placeholders such as `<server-ip>` instead of hard-coded IP
addresses in documentation.
