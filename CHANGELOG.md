# Changelog

## v0.9.1-alpha - In Progress

### Added

- Modular `smc` CLI.
- Pool manager foundation.
- Pool repository layer.
- BTC/BCH solo and public templates.
- Pool create/list/validate/enable/disable/remove/restore commands.
- Service management commands.
- Graph output for Nexus infrastructure mapping.
- REST API foundation.
- API systemd service.
- API readiness endpoint.
- API production hardening.
- Beta-free license/subscription foundation.
- Doctor command for install/readiness diagnostics.

### Changed

- API refactored into modular route structure.
- Status output now includes pools, health, and service state.

### Notes

- Current license edition is `beta-free`.
- Billing is disabled.
- Limits are currently unlimited for testers.
- Official beta OS targets are Ubuntu 24.04 LTS, Debian 12, and Debian 13.
