# Seymour MiningCore

Seymour MiningCore is a commercial mining pool platform built from MiningCore.

It is designed to run standalone with a Local Console or connect to Nexus Command Center for centralized mining infrastructure management.

## Core Features

- Solo pool hosting
- Public pool hosting
- Multi-coin roadmap
- Multi-pool roadmap
- Local Console UI
- Nexus Command Center integration
- REST API
- Installer scripts
- Health checks
- Mandatory 0.75% developer fee

## Nexus Command Center

Nexus Command Center is the optional management platform for monitoring and controlling one or many Seymour MiningCore servers.

Nexus repository:

https://github.com/imdmanuc2/nexus-command-center
