# Seymour MiningCore v0.9.0-alpha

**Release Date:** July 2, 2026

Welcome to the first official alpha release of **Seymour MiningCore**.

This release establishes the foundation for a modern, self-hosted
command center for solo cryptocurrency mining built on top of
MiningCore.

> **Status:** Pre-release (Alpha)\
> Intended for internal development and early testing.

------------------------------------------------------------------------

# Highlights

## Modern Command Center

-   Live Pool Dashboard
-   Responsive dark theme
-   Interactive charts
-   Fleet Health Score
-   Active Mining Operations
-   Educational tooltips

## Mining Analytics

-   Live Pool Hashrate
-   Per-Miner Hashrate
-   Probability Simulator
-   Multi-Coin Solo Lottery
-   Block Race Meter
-   Latest Block Preview

## Fleet Management

-   ASIC Fleet Overview
-   Worker Rankings
-   Live Worker Statistics

## User Experience

-   Hall of Fame
-   Achievement System
-   TV Mode

## Platform
