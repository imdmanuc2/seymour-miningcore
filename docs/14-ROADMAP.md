# 14 - ROADMAP

**Status:** Active\
**Owner:** Seymour MiningCore Team\
**Last Updated:** v0.9.1-alpha

## Purpose

This document describes the long-term direction of the Seymour
MiningCore platform.

It focuses on product capabilities and technical milestones rather than
release dates.

------------------------------------------------------------------------

# Current Stage

**Version:** v0.9.1-alpha

Current focus:

-   Stabilize the CLI
-   Complete REST API foundation
-   Finish documentation
-   Build production installer
-   Prepare for closed alpha testing

------------------------------------------------------------------------

# Phase 1 --- Foundation (Current)

Goals:

-   Modular `smc` CLI
-   REST API
-   Health monitoring
-   Infrastructure graph
-   Pool management
-   Licensing foundation
-   Local Console foundation
-   Documentation overhaul

Status: **In Progress**

------------------------------------------------------------------------

# Phase 2 --- Closed Alpha

Goals:

-   Production installer
-   First Boot Wizard (`smc setup`)
-   Nexus pairing (`smc pair`)
-   Token authentication
-   Improved diagnostics
-   Expanded health checks
-   Initial Local Console UI

Target outcome:

A small group of trusted testers can deploy and operate Seymour
MiningCore.

------------------------------------------------------------------------

# Phase 3 --- Public Beta

Goals:

-   Ubuntu and Debian validation
-   Additional coin templates
-   Enhanced pool management
-   Remote management through Nexus
-   Marketplace foundation
-   Plugin architecture
-   Performance improvements

Target outcome:

Public beta suitable for wider community testing.

------------------------------------------------------------------------

# Phase 4 --- Version 1.0

Goals:

-   Stable production release
-   Commercial licensing
-   Subscription support
-   Enterprise deployment guidance
-   Comprehensive documentation
-   Long-term maintenance process

Target outcome:

Production-ready mining platform.

------------------------------------------------------------------------

# Long-Term Vision

Seymour MiningCore will provide:

-   Professional mining pool management
-   Secure multi-server operations
-   Tight integration with Nexus Command Center
-   Extensible architecture
-   Modern management experience
-   Reliable upgrade path

------------------------------------------------------------------------

# Roadmap Principles

-   API-first development
-   Backward compatibility whenever practical
-   Security by default
-   Documentation maintained alongside code
-   Modular architecture
-   Stable release process
