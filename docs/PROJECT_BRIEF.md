# Seymour MiningCore Project Brief

## Overview

Seymour MiningCore is a commercial mining pool platform built on MiningCore.

### Goals

- Easy installation
- Solo pool hosting
- Public pool hosting
- Multiple coins
- Multiple pools per server
- Local Console UI
- REST API
- Nexus Command Center integration
- Mandatory 0.75% developer fee

## Product Split

### Seymour MiningCore
Runs on the mining server and includes:
- Mining engine
- Installer
- Pool manager
- Local Console
- REST API
- Health checks
- Update checks
- Nexus connector

### Nexus Command Center
Optional centralized management platform for:
- Multiple servers
- Remote installs
- Fleet monitoring
- Infrastructure map
- ASIC inventory
- Alerts
- Historical graphs
- AI assistant
